## Setup Instructions âš™ï¸

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ImAtharva4907/algo-Genius.git
   ```


   ```bash
   cd Backend && npm start
   ```

   ```bash
   cd ../Frontend && npm run dev
   ```
6. Open your browser and go to `http://localhost:3000`
